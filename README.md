# Homebrew Tap for ccrider

Homebrew formulae for ccrider - search, browse, and resume your Claude Code sessions.

## Installation

```bash
brew install neilberkman/ccrider/ccrider
```

## About

ccrider indexes your Claude Code sessions into SQLite with full-text search. Browse sessions in a polished TUI, search across all messages instantly, and resume any session with one keystroke.

See [github.com/neilberkman/ccrider](https://github.com/neilberkman/ccrider) for more information.
